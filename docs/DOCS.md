# Документация проекта Random Theme Selector

## Общее описание решения

Данный проект представляет собой расширение для Visual Studio Code, которое позволяет случайным образом выбирать и применять цветовые темы из числа установленных в редакторе. Расширение автоматически обнаруживает все доступные цветовые схемы и предоставляет простой интерфейс для их случайного выбора.

Проект реализован на TypeScript с использованием VSCode Extension API.

## Описание функций

### Модуль extension.ts

**Функция ```activate(context)```**

Активирует расширение при его загрузке в VSCode. Регистрирует команду для выбора случайной темы.

Параметры:

• ```context``` (vscode.ExtensionContext) - контекст расширения, предоставляемый VSCode

Возвращаемое значение:

• ```void```

Пример использования:
```typescript
export function activate(context: vscode.ExtensionContext) {
    const disposable = vscode.commands.registerCommand('random-theme.selectRandomTheme', () => {
        // логика выбора темы
    });
    context.subscriptions.push(disposable);
}
```

**Функция ```getInstalledColorThemes()```**

Собирает список всех установленных цветовых тем в системе VSCode.

Параметры:

• отсутствуют

Возвращаемое значение:

• ```string[]``` - массив идентификаторов всех найденных цветовых тем

Пример использования:
```typescript
const themes = getInstalledColorThemes();
// Возвращает: ['Default Dark+', 'Default Light+', 'Monokai', 'One Dark Pro', ...]
```

**Логика работы функции:**

1. Получает список всех установленных расширений через ```vscode.extensions.all```
2. Для каждого расширения проверяет раздел ```contributes.themes``` в package.json
3. Собирает все уникальные идентификаторы цветовых тем
4. Возвращает массив найденных тем

## Архитектурные особенности

### Работа с VSCode API

Расширение использует следующие компоненты VSCode API:

- ```vscode.extensions``` - для доступа к метаданным установленных расширений
- ```vscode.workspace.getConfiguration()``` - для изменения настроек темы
- ```vscode.window``` - для показа уведомлений пользователю
- ```vscode.commands``` - для регистрации команд расширения

## Требования к окружению

- Visual Studio Code версии 1.103.0 или выше
- Node.js 2.x (для разработки)
- TypeScript 5.9+ (для разработки)
